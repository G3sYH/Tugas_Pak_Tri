Seleksi Majemuk
1. Input nilai
2. Menampilkan grade A/B/C/D
3. Menampilkan lulus/tidak lulus
4. Selesai
Pseudocode
Start
Input nilai
if nilai >= 85 maka
grade = ‘A’
else if nilai >= 75 maka
grade = ‘B’
else if nilai >= 65 maka
grade = ‘C’
Output “Lulus”
else if nilai <= 65 maka
grade = ‘D’
Output”Tidak Lulus”
ENDIf
END
